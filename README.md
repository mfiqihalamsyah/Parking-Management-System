# Parking System

A simple parking management system built using C# and .NET. This application simulates vehicle entry, parking slot management, and exit processes.

## 🚀 Features

* Vehicle entry registration
* Parking slot allocation
* Vehicle exit tracking
* Basic parking logic system

## 🛠 Tech Stack

* C#
* .NET

## ▶️ How to Run

1. Open the project in Visual Studio
2. Build the solution
3. Run the application

## 💡 Notes

This project was created to practice backend logic and system design using C#.
